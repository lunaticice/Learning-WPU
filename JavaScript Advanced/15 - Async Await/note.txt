Asynchronous Function : Sebuah function yang bekerja secara asyncronous (melalui event loop) yang menghasilkan (implisit) promise sebagai return value-nya, tapi cara penulisa code-nya menggunakan penulisan yang syncronous(standard) | Sebuah async function dapat memiliki keyword await di dalamnya untuk memberhentikan sementara eksekusi fungsinya sambil menunggu promise-nya selesai.
